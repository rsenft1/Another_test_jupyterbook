How to make a google doc into a JupyterBook

  1. Install necessary software

  1. Docs to Markdown Google Add-on
  2. pip install -U jupyter-book

  1. Make sure your google doc is organized

  1. H1 headings will become separate pages for the JupyterBook
  2. Example structure (pages will be Home, 1. Install DeepProfiler, etc.):
  4. How to indicate code? Courier new or monospaced fonts
  5. How to indicate notes? (also warning, tip, important) - check out next section

  1. Convert the Google doc to markdown

  1. Go to Add-ons > Docs to Markdown > Convert
  2. Check the boxes for Suppress info comment and Use reckless mode
  3. Hit the blue “Markdown” button

  1. Paste into Atom and save as markdown file (.md)

  2. Run my script to generate JupyterBook files

  1. md2Book(filepath, delim="\n# ", bookName="mybook", numbered=False, savePath="same", makeTOC=True, Landing="Home")
  1. Download and fix image refs
  1. Download images from Google doc File > Download > Web page
  2. Images will have names where the number doesn’t correspond to order. Change the image files to be correct
  3. Images should be located in docs/images/ and should have names like “image1.png”

  1. Add a config file

  1. Should be added in /mybook folder

  2. Here’s a good example: Configuration reference 

  1. Build the book

  1. In terminal, navigate to book
  2. jupyter-book build --all mybook/
